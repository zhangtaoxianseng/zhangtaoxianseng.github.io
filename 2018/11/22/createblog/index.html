#搭建开始

1.新建一个Repository

在repository name下填写 yourname.github.io

创建成功


建立一个以你自己名字 yourname.giothub.io 域名地址

2.安装github

安装到自己电脑终端不会自己百度，还有安装node环境

3.安装hexo环境

在你的终端里安装

$ npm install -g hexo-cli

4.初始化hexo

进入你要初始化hexo的文件里，
$ hexo init blog
就可以初始化博客文件

$ hexo server
可以本地运行，预览博客内容

5.为 Hexo 安装 Git 插件

安装 hexo-deployer-git，否则会报 ERROR Deployer not found: git 的错误。

$npm install hexo-deployer-git –save

修改你的 _config.yml 配置文件如下：

Deployment

Docs: https://hexo.io/docs/deployment.html

deploy:
type: git
repo: https://github.com/yourname/yournaem.github.io.git
branch: master

6.生成静态文件和部署

$ hexo g

$ hexo d

最后出现以下提示代表成功了

INFO Deploy done: git

你就可以打开以自己名字命名的yourname.github.io的博客了
