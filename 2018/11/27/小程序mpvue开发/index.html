小程序mpvue开发

2018-11-27  

mpvue （github 地址请参见）
是一个使用 Vue.js 开发小程序的前端框架。框架基于 Vue.js 核心，mpvue 修改了 Vue.js 的 runtime 和 compiler 实现，使其可以运行在小程序环境中，从而为小程序开发引入了整套 Vue.js 开发体验。

1. 快速上手

在node环境下开发，安装git和node环境

打开终端命令行工具：

   # 1.先检查下node.js 是否安装成功
$ node -v
v8.9.0

$ npm -v
5.6.0

# 2.由于众所周知的原因，可以开旭切换源为taobao源
$ npm set registry https://registry.npm.taobao.org/

# 3.全局安装 vue-cli
# 一般是要 sudo 权限 的
$ npm install --global vue-cli@2.9

# 4.创建一个基于 mpvue-quickstart 模板的新项目
# 新手一路回车选择默认就可以了
$ vue init mpvue/mpvue-quickstart my-project

# 5.安装依赖，运行
$ cd my-project
$ npm i
$ npm run dev

运行成功的后，可以看到本地目录下多了个 dist 文件，里面是生成的小程序代码。

2. 搭建小程序的开发环境

微信小程序需要用 微信开发者工具 开发测试

3. 调试开发mpvue

打开微信开发者工具，填写里面的项目信息

此时mpvue项目测试成功，用你自己的代码编辑器去编辑 my-project 中的 src 目录下的代码

4. 分包机制

mpvue-loader 1.1.2-rc.2之后，优化了build后的文件生成结构，生成的目录结构保持了源文夹下的目录结构，有利于分包的支持。

注意事项

新增的页面需要重新 npm run dev 来进行编译
